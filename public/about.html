About - 표준화 달력 프로젝트

이 웹 애플리케이션은 표준화 작업을 하는 분들의 일정 관리를 돕기 위해 개발되었습니다. 회의 날짜와 각 회의 유형(정기회의, RGM)에 따른 기고서 마감일을 쉽게 확인할 수 있도록 설계되었습니다.

프로젝트 목적

표준화 업무를 진행하는 과정에서 회의 일정과 문서 제출 기한을 쉽게 파악하는 것은 매우 중요합니다. 이 애플리케이션은 사용자가 회의 날짜를 선택하면 자동으로 각 회의 유형별 기고서 마감일(정기회의: 12일 전, RGM: 7일 전)을 계산하여 보여줌으로써, 일정 관리의 편의성을 높이고 마감일을 놓치지 않도록 도와줍니다.

주요 기능

  • 2개월 달력 한눈에 보기 - 현재 월과 다음 월의 달력을 동시에 표시하여 일정 계획을 용이하게 합니다.
  • 회의 날짜 선택 기능 - 달력에서 회의 날짜를 클릭하여 선택할 수 있으며, 선택한 날짜에 대한 상세 정보가 표시됩니다.
  • 정기회의 기고서 마감일 자동 계산 - 선택한 회의 날짜로부터 12일 전 날짜를 자동으로 계산하여 정기회의 기고서 마감일로 표시합니다.
  • RGM 기고서 마감일 자동 계산 - 선택한 회의 날짜로부터 7일 전 날짜를 자동으로 계산하여 RGM 기고서 마감일로 표시합니다.
  • 마감일 상태 알림 - 마감일이 지났거나, 오늘이 마감일이거나, 마감일이 3일 이내로 남은 경우 경고 메시지를 표시합니다.
  • URL 파라미터를 통한 날짜 공유 - 선택한 날짜 정보가 URL에 반영되어 특정 날짜 정보를 다른 사용자와 공유할 수 있습니다.
  • 키보드 네비게이션 - 좌우 화살표 키를 사용하여 이전/다음 달로 쉽게 이동할 수 있습니다.

이 프로젝트는 Node.js의 http 모듈을 사용하여 구축되었으며, 순수 JavaScript를 활용하여 달력 기능을 구현했습니다. 모든 처리는 클라이언트 측에서 이루어져 서버 부하가 적고 응답 속도가 빠릅니다.

개발 배경

표준화 작업을 진행하는 많은 전문가들이 다양한 회의와 문서 제출 일정을 관리하는 과정에서 어려움을 겪고 있습니다. 특히 여러 표준화 기구의 회의가 겹치는 경우, 각 회의별 마감일을 계산하는 것은 번거로운 작업입니다. 이 도구는 그러한 불편함을 해소하고, 사용자가 중요한 마감일을 놓치지 않도록 돕기 위해 개발되었습니다.

사용 방법

  1. 메인 페이지의 달력에서 회의 날짜를 클릭합니다.
  2. 선택한 날짜에 대한 정보가 하단에 표시됩니다.
  3. 정기회의 기고서 마감일(12일 전)과 RGM 기고서 마감일(7일 전)이 자동으로 계산되어 표시됩니다.
  4. 달력에서 마감일이 색상으로 표시됩니다 (정기회의: 오렌지색, RGM: 분홍색).
  5. 왼쪽/오른쪽 화살표 버튼을 클릭하거나 키보드의 화살표 키를 사용하여 이전/다음 달로 이동할 수 있습니다.

앞으로도 표준화 업무를 지원하기 위한 다양한 기능이 추가될 예정입니다. 기능 개선 제안이나 피드백은 언제나 환영합니다.
